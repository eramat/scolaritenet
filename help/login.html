Si vous connectez pour la première fois et que vous avez un compte alors saisissez votre login et validez. Il vous faudra ensuite indiquer un mot de passe. Après confirmation de ce dernier, votre compte devient actif et vous êtes de nouveau sur la fenêtre d'identification.

Après validation de votre login et de votre mot de passe, deux cas de figure peuvent se présenter : soit vous appartenez qu'à un seul type d'utilisateur (étudiant, par exemple) soit vous possédez plusieurs fonctions et dans ce cas, il vous sera proposé de sélectionner le type d'utilisateur avec lequel vous accédez è l'application.
Vous pouvez accéder à l'application seulement en tant qu'un seul type d'utilisateur. Si vous voulez changer alors il faut de déconnecter et se reconnecter.
